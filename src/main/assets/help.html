Frequently Asked Questions

How to shuffle card set?
Select menu Shuffle. The lesson will start from random card of the card set.
To get back to initial card set sequence use the same menu item.
How to mark a card
If you couldn't answer card question correctly, you can mark a card by long tap on the card text. The text of the marked card will be highlighted in orange color. To unmark a card just use long tap on the card text again.
How to show marked cards only?
If you marked some cards of the particular lesson, you can use menu Show marked to work only with marked cards.
After switching to this mode the card set will contain only marked cards.
To get back to initial card set use the same menu item.
How to show card back side (answer) first?
There is an option to show card back side first. Use menu Show back first. After selecting this option every next card will show back side text first.
To get back to default (show face first) mode use the same menu item.
